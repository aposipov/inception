This project aims to broaden my knowledge of system administration by using Docker. I will virtualize several Docker images, creating them in my new personal virtual machine.

In This Project

Nginx

Nginx Container

A Docker container that contains NGINX with TLSv1.2 or TLSv1.3 only.
Wordpress

WordPress Container

A Docker container that contains WordPress + php-fpm only without nginx
Mariadb

Mariadb Container

A Docker container that contains MariaDB only without nginx.

Bonus part:

Redis Caches

Redis Cache Container

A redis cache for your WordPress website in order to properly manage the cache.
Adminer

Adminer Container

A Docker container that contains Adminer runnig on port 8080.
FTP

vsftpd Container

A Docker container that contains vsftpd runnig on port 21.
Website

Website Container

A Docker container that contains Website runnig on port 8888.
cAdvisor

cAdvisor Container

A Docker container that contains cAdvisor runnig on port 8090.
